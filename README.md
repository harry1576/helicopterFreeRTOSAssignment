# Tiva CMake (Now with FreeRTOS)
Yay! CCS is no more CMake is here to save you.

## Building
```bash
user@linux~$ mkdir build
user@linux~$ cd build
user@linux~$ cmake ..
user@linux~$ make
```
## Flashing
```bash
user@linux~$ make flash
```
